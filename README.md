# Stock-Indicator-Monitor

一个监控A股个股技术指标的纯Python应用

# Background
东方财富、同花顺等平台提供的**指标监控**功能和**数据分析**功能有限，无法充分满足个性化的指标跟踪和辅助交易决策的需求。因此决定开发本项目，用于辅助获取**交易性**的投资机会。

另外，**结构性**的投资机会需要结合**宏观政策**和**公司基本面**分析，借助大模型Agent理应能取得一定效果，但不在本项目的范围内。
